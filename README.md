# unattackable

unattackable小组实现的基于tensorflow的恶意访问识别算法，该需求来自于2017云计算大赛命题四。


## Requirements
- Python 2.7.X
- TensorFlow >= 1.2


To train:

```
python model.py \
  [--data_path access.log]
```

To sample:

```
python sample.py \
  [--data_path access.log]
```
